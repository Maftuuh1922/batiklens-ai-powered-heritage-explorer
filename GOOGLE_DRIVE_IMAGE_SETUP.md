# Setup Google Drive untuk Hosting Images

## Langkah 1: Upload ke Google Drive

1. Buka Google Drive
2. Buat folder baru: `batik-images`
3. Upload seluruh folder `Dataset_Batik_Asli` ke dalam folder `batik-images`
4. Klik kanan pada folder `batik-images` → Share → Anyone with the link can view
5. Copy link sharing

## Langkah 2: Cara Dapatkan Direct Link

Dari sharing link:
```
https://drive.google.com/drive/folders/FOLDER_ID?usp=sharing
```

Untuk direct image:
```
https://drive.google.com/uc?export=view&id=FILE_ID
```
